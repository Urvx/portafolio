Explicación de API

La API utilizada para está práctica es la API Webstorage, la cual sirve
para almacenar temporalmente datos de manera local que puedan ser
introducidos por el usuario que utilice la página.

Importancia de la API

La API Webstorage es de las API's más utiñiz al momento de realizar
una página en internet, sobre todo en el caso de páginas que tengan
sistema de logeo, ya que esta puede guardar los datos de todos los
usuarios y eliminarlos si se le permite a los mismo, así como otros
valores que se puedan requerir en cada tipo de página especial, que
requiera interacción con el usuario e información extra.

Utilidad en la página

En la página actual, la API Webstorage se utiliza como un registro de
usuario, el cual, con su nombre y matricula, se logeoa a través de una
solicitud de datos y estos se quedarán guardados mientras el usuario
este presente en la página.
